# KB Digitals - Cashback Rewards Platform

A modern, responsive web application that showcases popular brands and retailers where users can earn cashback rewards on their purchases. Built with React and featuring an intuitive interface with featured brand carousels and comprehensive brand listings.

## Features

- **Featured Brands Carousel**: Interactive swiper carousel highlighting top cashback offers
- **Brand Directory**: Grid layout displaying all available brands with cashback percentages
- **Boosted Offers**: Special highlighted deals with increased cashback rates
- **Contact Form**: User-friendly contact section for inquiries and support
- **Responsive Design**: Fully responsive layout optimized for mobile, tablet, and desktop devices
- **Modern UI**: Clean, professional interface built with TailwindCSS

## Tech Stack

- **React 19** - UI library
- **Vite** - Fast build tool and development server
- **TailwindCSS 4** - Utility-first CSS framework
- **Swiper** - Modern touch slider for carousels
- **React Icons** - Icon library
- **ESLint** - Code linting and quality

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Bhavna-jsr/kbdigitals.git
cd kbdigitals
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start development server with hot reload
- `npm run build` - Build for production
- `npm run preview` - Preview production build locally
- `npm run lint` - Run ESLint to check code quality
